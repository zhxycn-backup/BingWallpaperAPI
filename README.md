# BingWallpaperAPI
基于 PHP 的必应壁纸 API。

## 使用前的准备
下载代码并部署到服务器或克隆代码。
```
git clone --recurse-submodules https://github.com/zhxycn/BingWallpaperAPI.git
```

### 设定图片的时间
`index.php` 第 10 行 `$ago = '0'; `

`''`中的数字意为几天前。详见下表。

数字|含义
-|-
-1|明天
0|今天
1|昨天
2|前天
……|……

默认为`0`，以此类推。范围`-1 ~ 7`。由于调用的是必应国内版，当日16至次日16时为一张壁纸。

## 参数说明
参数名|含义
-|-
encode|返回数据格式

>不加参数则直接返回图片。

### 数据格式说明
参数值|含义
-|-
json|返回 JSON 格式数据

## 图片参数
分辨率 `1920x1080`
